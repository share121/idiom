# Idiom Library

## brief introduction

Contains numerous idioms

## Usage method

All idioms are saved in `src/idiom.json`, and you can copy this json file into your project

## Secondary development

Modify the content in `src/index.ts`, use `yarn build` to generate `src/index.js`, and then run this JavaScript file
You can directly compile and run using the `yarn start` command
You can use `yarn start -h` to view more options

## Idiom sources

- [成语大全 - 千篇国学](https://chengyu.qianp.com/)
- [百度汉语 - 更懂汉语，更懂你](https://hanyu.baidu.com/)
